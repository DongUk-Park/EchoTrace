# EchoTrace

![Framework Architecture](./assets/overview.png)

This is the official implementation of the paper **Echoes in the Loop: Diagnosing Risks in LLM-Powered Recommender Systems under Feedback Loops** (https://www.arxiv.org/abs/2602.07442).

---
⚠️ **Repository Status: Refactoring in Progress**

The current codebase is a direct export of the experimental environment used for the paper. 
We are currently refactoring for better readability and modularity. 
- **Current Version:** Functional but experimental.
- **Estimated Clean Update:** Late Feb 2026.
